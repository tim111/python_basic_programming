Maak een woordspelletje waarbij de gebruiker op volgende drie vragen moet beantwoorden:

Wat is je lievelingsdier?
Wat is je lievelingseten?
Geef een getal kleiner dan twintig

De antwoorden worden telkens ingegeven door de gebruiker.

Invoer

  • Wat is jouw lievelingsdier?
  • Wat is je lievelingseten?
  • Geef een getal kleiner dan twintig:

Uitvoer

Daarna toont het programma een gekke zin met de antwoorden

Voorbeeld

Invoer:

olifant
frieten
8

Uitvoer:

Ik droom altijd van een olifant met 8 poten die frieten eet.
