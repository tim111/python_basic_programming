Gegeven een natuurlijk getal.

Schrijf een programma dat het antwoord is op de vraag “Bestaat het ingevoerde getal uit exact 3 cijfers?”.

Invoer

Een natuurijk getal.

Uitvoer

2 mogelijkheden
Het getal ... bestaat uit 3 cijfers.
Het getal ... bestaat niet uit 3 cijfers.

Voorbeeld

Invoer:

Bestaat het ingevoerde getal uit exact 3 cijfers? 113

Uitvoer:

Het getal 113 bestaat uit 3 cijfers.

Voorbeeld 2

Invoer:

Bestaat het ingevoerde getal uit exact 3 cijfers? 100025

Uitvoer:

Het getal 100025 bestaat niet uit 3 cijfers.
