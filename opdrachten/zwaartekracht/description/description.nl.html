Door het verschil in zwaartekracht op elk hemellichaam is je gewicht nergens hetzelfde als op Aarde. Op sommige hemellichamen is de zwaartekracht groter en weeg je dus meer, op andere is de zwaartekracht kleiner en weeg je weer minder.
Naar welk hemellichaam verhuis jij het best om je ideaal gewicht te bekomen? Voer je gewicht in, kies een hemellichaam en voer je massa op het gekozen hemellichaam uit.

  • je massa op de maan is je gewicht x 0,166
  • je massa op Jupiter is je gewicht x 2,36
  • je massa op Mars is je gewicht x 0,37
  • je massa op Venus is je gewicht x 0,9
  • je massa op Neptunus is je gewicht x 1,12

Het hemellichaam moet correct worden ingegeven: maak een onderscheid tussen een hoofdletter en een kleine letter. Voor elk ander hemellichaam of foutieve schrijfwijze geef je een foutboodschap weer.

Voorbeeld 1

Invoer:

Geef je gewicht in (kg): 65.3
Op welk hemellichaam ben je? Mars

Uitvoer:

Op Mars weeg je 24.161 kilogram..
