We bevinden ons in het casino van Knokke. Maak een eenvoudige jackpot. Een speler kan op voorhand zijn/haar inzet bepalen. De jackpot toont drie willekeurige getallen tussen één en vijf. Afhankelijk van deze getallen wordt de score bepaald.

Bij drie gelijke getallen is de score drie maal de inzet. Bij twee gelijke getallen is de score twee maal de inzet. Bij drie verschillende getallen is de score de negatieve waarde van de inzet.

Invoer

Geef jouw inzet.

Uitvoer


De 3 willkeurige getallen zijn x | x | x
Proficiat, je verdient 3 x jouw inzet namelijk ... euro.
Proficiat, je verdient 2 x jouw inzet ... euro.
Helaas je verliest jouw inzet ... euro.

Geef jouw inzet: 20

Uitvoer:

De 3 willekeurige getallen zijn 3 | 3 | 1
