Bij het ingeven van je PIN-code aan een geldautomaat, heb je drie pogingen om je correcte PIN-code in te geven. Na drie mislukte pogingen wordt de chipkaart geblokkeerd.

Schrijf een Python script dat een gebruiker vraagt om een PIN-code in te geven. Indien deze correct is wordt het saldo van zijn/haar rekening getoond. Na drie mislukte pogingen krijgt de gebruiker de melding bankkaart geblokkeerd! en wordt het programma beëindigd. Volgende stappen voer je uit:

  • Kies een ‘geheime’ PIN-code bv. pin = "4567" en een saldo, bv. saldo = 1223.5
  • 2Maak een variabele aan die het aantal ‘resterende’ pogingen bijhoudt (initieel drie)

Een mogelijke input/output is:

Typ uw PIN - code : 3258
PIN - code incorrect ! Aantal resterende pogingen : 2
Typ uw PIN - code : 0066
PIN - code incorrect ! Aantal resterende pogingen : 1
 Typ uw PIN - code : 1258PIN - code incorrect ! Aantal resterende pogingen : 0
Bankkaart geblokkeerd !
Typ uw PIN - code : 4567
PIN - code correct ! Uw saldo is 1223.5 Euro.
