Schrijf een programma dat de ideale massa van de gebruiker berekent.

Invoer

  • Je vraagt de lengte van de gebruiker op (in cm).
  • Je vraagt de polsomtrek van de gebruiker op (in cm). Let op dit kan een decimaal getal zijn.
  • Je berekent de ideale massa aan de hand van volgende formule: (lengte - 100 + 4 * polsomtrek) / 2

Uitvoer

De ideale massa van de gebruiker.

Voorbeeld

Invoer:

Geef jouw lengte op:181
Geef jouw polsomtrek op:10.4

Uitvoer:

Jouw ideale massa is 61.3
