Schrijf een programma dat de gebruiker om zijn/haar voornaam vraagt.

Daarna begroet het programma de gebruiker door “Hallo, voornaam!” op het scherm te tonen.

Het programma...

  • vraagt de gebruiker naar een (voor)naam en kent deze waarde toe aan een variabele (bv. voornaam).
  • stelt een nieuwe ‘string’ samen.
  • drukt de nieuwe string af op het scherm.

Tip: Gebruik het plusteken (+) als operator om strings samen te voegen (concatenation)
