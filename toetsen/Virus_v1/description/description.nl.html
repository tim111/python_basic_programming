Het laboratorium onderzoekt met welke virusstam een persoon besmet is. Er zijn drie virusvarianten in omloop:

  • de oorspronkelijke Wuhan-variant (W);
  • de besmettelijke Britse variant (B);
  • de gevaarlijke Zuid-Afrikaanse variant (Z).

Op het einde van elke dag wil men over de volgende gegevens beschikken:

  • Een lijstje met alle virus-varianten.
  • Omdat de Britse variant de meest besmettelijke is, wil men het aantal stalen van deze variant kennen.

Je vraagt aan de gebruiker per staal de ontdekte variant op. Wanneer je voor alle stalen de variant één voor één opgegeven hebt, sluit je af met een #.
Je geeft als uitvoer alle opgegeven varianten in een string en het aantal 'gevonden' varianten.

Maak een nieuwe python file. Sla het bestand op als jouwnaam_seminarie_virus.py. Schrijf de nodige programmacode. Debug indien nodig en drop het .py-bestand in de uploadzone van het seminarie

Invoer:

W
B

B
B
#
