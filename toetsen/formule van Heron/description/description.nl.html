De formule van Heron is een bijzondere formule waarmee de oppervlakte van een driehoek berekend kan worden aan de hand van de zijden.
De formule is genoemd naar Heron van AlexandriÃ« die deze bewezen heeft in zijn grote werk Metrica.

Opgave

Bereken de oppervlakte van een driehoek als de lengte van de 3 zijden ingelezen wordt. Geef het resultaat afgerond op 2 cijfers na de komma.

Maak een nieuwe python file. Sla het bestand op als jouwnaam_formule_van_heron.py. Schrijf de nodige programmacode. Debug indien nodig in jouw IDE en dien in op Dodona. Drop als back-up jouw py-bestand in de uploadzone van Smartschool.
