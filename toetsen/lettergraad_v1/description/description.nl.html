Schrijf een Python script dat een lettergraad toekent aan een bepaalde percentage-intervallen

  • A voor [90, 100]
  • B voor [80, 90[
  • C voor [70, 80[
  • D voor [60, 70[
  • Geen lettergraad indien percentage < 60

Het programmaatje vraagt naar het percentage en schrijft de lettergraad naar het scherm. Een percentage boven de 100% is onmogelijk.

Je vraagt aan de gebruiker zijn/haar percentage. De invoer gebeurt op een nieuwe regel.

Je geeft als uitvoer de bijhorende graad. Bijvoorbeeld: Je krijgt de graad A.

Maak een nieuwe python file. Sla het bestand op als jouwnaam_virus.py. Schrijf de nodige programmacode. Debug indien nodig en dien in op Dodona.

Invoer:

Wat is jouw percentage ?
70

Uitvoer:

Je krijgt de graad C.
