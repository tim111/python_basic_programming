Bereken het totaal terug te betalen bedrag en totale betaalde rente

  • Vraag aan de gebruiker hoeveel cupcakes er worden gekocht.
  • Berekende de totale prijs van de cupcakes.
  • Toon het bedrag in een correcte zin.

Maak een nieuwe python file. Sla het bestand op als jouwnaam_cupakes.py. Schrijf de nodige programmacode. Debug indien nodig in jouw IDE en dien in op Dodona. Drop als back-up jouw py-bestand in de uploadzone van Smartschool.
