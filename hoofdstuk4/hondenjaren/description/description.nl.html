Vraag aan de gebruiker de leeftijd van zijn hond. Nadien print je een correcte boodschap waarbij:

  • Indien getal < 0, dan geef je een foutmelding terug.
  • Indien leeftijd = 1 jaar -> 14 mensenjaren.
  • Indien leeftijd = 2 jaar -> 22 mensenjaren.

Leeftijd klopt niet!

Uw hond is 14 hondenjaren oud.
